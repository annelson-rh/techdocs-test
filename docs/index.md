## What is TechDocs?

TechDocs is the built-in docs-as-code solution in [Backstage](https://backstage.io) (and Red Hat Developer Hub). It uses [MkDocs](https://www.mkdocs.org/) under the hood to generate static documentation from Markdown files in your repository.

**Key benefits:**

- Documentation lives next to the code it describes
- Changes go through the same review process as code
- Automatic publishing — no separate deploy step
- Searchable and integrated into the Developer Hub catalog
